# docs/assets

Place image and media assets here. Referenced from README.md and other docs.

## Required assets (Phase 1 follow-ups)

### `banner.png`
- **Dimensions:** 1280x640 (2:1)
- **Format:** PNG, under 500KB
- **Use:** social card, top of README (optional)
- **Content:** tracebloom wordmark on `--bg` (`#0B0D10`), one-line tagline.

### `demo.gif`
- **Dimensions:** 1200x675 (16:9)
- **Format:** GIF, under 5MB
- **Duration:** 12–20 seconds, looped
- **Use:** hero GIF in README
- **Content:** CLI run → trace recorded → viewer opens → user clicks through a flame graph → finds the bad decision. No voiceover, no captions — the motion should read on its own.

Both assets must be self-hosted (no CDN) and placed in this directory.
